<Update label="2025-04-10" description="Java SDK 0.9.0 发布">

- 降低 JDK 要求至 JDK8
- 添加 Claude Desktop 集成示例
- 完整更新日志可在此查看：https://github.com/modelcontextprotocol/kotlin-sdk/releases/tag/0.4.0
